Special use of Terminal
by Eli Lien – tags: #terminal, #mac, #tips

用 Terminal 可以简单地完成很多事情，在我看来，比 OS X GUI 还要来的方便。

  1. lsof

    列出当前用户所打开的文件以及打开这些文件对应的进程等，当 OS X 提示 "The item can't be xxx because it's xxx" 时可使用该命令查看使用该文件的进程并 kill 掉。

  2. open

    该命令几乎可以打开一切我们所能想到的东西，文件/文件夹，网址 (open http://elilien.com)，远程屏幕共享 (open vnc://1.2.3.4)，甚至查找单词 (open dict://apple)。

  3. pbcopy 与 pbpaste

    这两个命令能在 Terminal 与 OS X Clipboard 之间复制/粘贴数据，使用 lsof | pbcopy 可以将 lsof 当前内容复制到 OS X Clipboard 中，在需要之处用 ⌘+V 粘贴。

  4. rm

    删除命令。可以在一瞬间批量删除多个文件/文件夹，名称支持通配符 "*"，慎用。rm -i 可在删除文件/文件夹前询问用户。查看更多用法 man rm。

  5. killall appname / kill PID

    强制退出应用/进程 (当应用无响应的时候)。

  6. sudo defaults write /Library/Preferences/com.apple.loginwindow LoginwindowText "Your Text"

    定制登录界面的提示，可以试试这个
    loginText="iSight image capture at login activen"`date "+on %m/%d/%Y at %H:%M:%S"`

  7. FORCE EMPTY THE TRASH

    • sudo rm -r 将 Trash 内所有文件拖拽到 Terminal 窗口然后 return (注意 "-r" 后有一空格)。
    • sudo rm -rf ~/.Trash/ /.Trashes/ /Volumes//.Trashes/
